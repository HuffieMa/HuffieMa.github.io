双系统Windows访问Ubuntu文件
发表于2024-03-10|实用工具Windows工具|实用工具
由于 Windos 和 Ubuntu 双系统在使用过程中，难免会遇到在使用一个系统时，会需要查看存放在另一个系统中的文件。 而在 Ubuntu 中可以直接挂载 Windows 中的磁盘来实现文件访问，但是 Windows 却没有提供这样的功能。 本文推荐一款软件 Ext2Fsd 可以将整个 Ubuntu 分区作为一个磁盘挂载在 Linux 下。 Ext2Fsd 下载 Ext2Fsd 可以用来读取 ext 格式的硬盘，经过网友测试，推荐下载 v0.68 版本，以下是下载链接： https://sourceforge.net/projects/ext2fsd/files/Ext2fsd/0.68/ 下载其中的Ext2Fsd-0.68.exe 下载完成后安装即可。 Ext2Fsd 使用 使用方法非常简单，只需要选中 Ubuntu 所在的分区，然后右键 加载装配点盘符 即可。 此时会发现系统中多了一个盘，可以读写 Linux 中的所有文件 注意，Ubuntu 系统文件读写务必谨慎操作。
