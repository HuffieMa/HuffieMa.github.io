  • 复杂环境下自主采样机器人 2023.03 ~ 至今

    • 项目描述：

       由于物体表面的不规则性和光照条件的变化，单一视角获得的点云通常在物体的边缘处会出现较大的空洞和误差，这可能导致不准确的抓取姿态估计。 为了解决此问题，提出了一种基于6D姿态估计和点云融合的实用机器人抓取方法。 首先进行物体6D姿态估计，利用模型点云和姿态估计的结果，通过ICP点云配准对齐输入点云和模型点云。 然后通过抓取姿态估计网络，利用得到的完整物体点云来估计物体的6D抓取姿态。

    已完成工作：

    1. 实现了一种基于视角匹配的无模型 6D 位姿估计网络，构建并训练位姿估计网络模型， 实现对已知物体的精准位姿估计，以及对未训练物体的粗略位姿估计方法。建立位姿优化网络，在已有稀疏点云的情况下实现未训练物体的位姿估计优化。

    2. 提出了一种基于位姿估计的物体点云融合与滤波方法， 针对机器人抓取样品时，单一视角在物体边缘处的点云空洞问题，设计点云融合方法利用位姿估计的输出对空洞补全，并对噪点进行滤除。

    3. 改进了基于点云的抓取姿态估计算法，在抓取角度预测网络与快速搜索策略的基础上，利用被补全的点云实现更稳定的抓取姿态生成，提高了抓取的成功率和鲁棒性。

    4. 在仿真和真实情况下进行了实验测试，使用了真实6自由度机器人与Realsense相机，上位机使用笔记本(ROS/Ubuntu20.04)

  • 腹腔镜手术自主缝合机器人系统 2023.02 ~ 2023.07

    • 项目描述:

    本项目旨在提高内镜手术中自主缝合的效率和准确性。包括3DOF自主缝合器械的设计， 开发一种基于3D腹腔镜的高效腹腔镜手术机器人，并建立一个集成视觉、专家和导航系统的机器人系统。

    已完成工作：

    1. 实现手术器械与病变组织识别，基于Yolov8对手术器械与病变区域进行实例分割，确定手术器械末端与病变区域的空间位置。

    2. 构建视觉与机械臂融合系统，进行相机与机械臂的手眼标定，实现视觉伺服的机器人简单动作

  • 岩心箱搬运机器人 2022.08 ~ 2023.02

    • 专利（实质审查）

    项目描述：

    从石油开采前的钻探中获得的岩心样本是评估采矿价值的关键数据，岩心样本一般存储在专用的岩芯箱中。 在研究和分析过程中，必须按照指定的顺序将这些岩芯箱整齐地布置在现场。 然而，由于岩心箱重量较大导致劳动强度很高，这会影响工作效率以及损伤工作人员。 本项目旨在给出一种搬运岩心箱的更高效地方法。

    已完成工作：

    1. 搭建了一套传感系统，利用红外传感器进行周围障碍物检测，以及利用面阵激光传感器检测当前码垛高度与整齐程度。

    2. 实现基于Mask RCNN的岩心箱实例分割，结合深度相机确定其边角空间坐标，用于机器人搬运过程中的视觉伺服。

    3. 实现机器人的原地位置2D姿态调整，通过识别Aruco的偏移距离和角度，利用单目相机与Aruco标记

  • 基于双目视觉的医用多自由度云台系统设计 2022.01 ~ 2022.06

    • Github | Demo Video

    项目描述:

    在传统的医疗手术过程中，医生往往需要经常变换位置调整角度来观察手术区域。 此外，在康复治疗期间，传统相机的监控范围经常不足以覆盖患者的活动区域。 目前常用的手术或康复过程监控方法具有较大的局限性并且自动化程度较低。 本项目设计了一种多自由度云台系统，用于跟踪医生和患者的位置。

    已完成工作：

    1. 多自由度云台的机械结构设计以及运动仿真，利用内齿式回转支承轴承尽可能缩小了结构尺寸，并对所有的传感器件进行了保护结构设计

    2. 设计了一套基于STM32与A4988驱动器的云台硬件控制算法，并实现了STM32与上位机ROS系统的通信，实现基于ROS的云台运动控制

    3. 提出了一种基于HOG与SIFT特征匹配的目标跟踪方法，实现了短期单目标跟踪的功能，并且对于目标物体的外观变化具有一定的鲁棒性

    4. 搭建了实验装置，并对所提出的所有算法进行了实验，使用3D打印结构件与Realsense相机以及42#步进电机

  • 面向广域信息采集的球形变胞机器人系统 2020.11 ~ 2021.05

    • Utility Model Patent

    项目描述：

    在石油化工场所、危化品仓库等场所实时监测各种环境信息对防灾减灾至关重要。 为了实现这一目的，我们设计了一个智能球形环境监测机器人系统。 它具有球形变胞结构和多个伸缩脚，使机器人能够实现高稳定性以及全方位运动。 机器人通过自组网通信技术实现多种环境信息监控模式， 并将收集到的数据传输到上位机系统，实现对大范围复杂环境信息的实时动态监测。

    已完成工作：

    1. 球形机器人的三维结构设计，包括球体内部控制器与传感器的布置，腿部电机的结构配置

    2. 基于STM32实现球形机器人腿部电机的控制，计算运动策略并实现机器人的运动

    3. 将蓝牙模块、加速度计、温湿度传感器、燃气传感器等集成到STM32控制器上，实现蓝牙控制机器人运动以及接收传感器数据

  • 身份识别与体温检测装置 2020.06 ~ 2021.10

    • 项目描述：

    为了应对疫情期间对非接触式体温测量和身份识别功能的需求，我们开发了相应的设备。 该系统以STM32单片机为主控板，利用红外温度传感器模块采集温度数据。 通过OpenMV模块实现人脸信息识别和口罩检测。

    已完成工作：

    1. 利用OpenMV相机实现人脸识别与口罩佩戴检测功能

    2. 基于STM32将相机、红外温度传感器、显示屏进行集成控制，形成一套身份识别与体温检测装置
